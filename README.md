# Pokédex Search+ Industries

Uma aplicação completa de consulta Pokémon que integra múltiplas APIs para fornecer dados técnicos, traduções automáticas e análises estatísticas comparativas.

## 🚀 Funcionalidades

- **Busca Avançada:** Consulta de dados via PokéAPI.
- **Tradução Automática:** Descrições traduzidas em tempo real utilizando a API MyMemory.
- **Simulador de Batalha:** Exibição dinâmica de oponentes aleatórios baseados nas vantagens e fraquezas de tipo do Pokémon pesquisado.
- **Análise Estatística:** Cálculo em tempo real da média de status de 5 Pokémons aleatórios do mesmo tipo para comparação com o Pokémon selecionado.
- **Interface Responsiva:** Layout "Arena" com três colunas, adaptável para dispositivos móveis.
- **Sugestões Inteligentes:** Sistema de sugestões de Pokémons semelhantes após cada busca.

## 🛠️ Tecnologias Utilizadas

### Back-end
- **Node.js**
- **Express** (Servidor HTTP)
- **Axios** (Requisições de API)
- **CORS** (Segurança e compartilhamento de recursos)

### Front-end
- **HTML5**
- **CSS3** (Grid Layout e Flexbox)
- **JavaScript Vanilla** (Consumo de API e manipulação de DOM)
